Изнутри frame необходимо получать информацию о доступных hackrf

это можно сделать с помощью hackrf_devie_list()

полученный список надо отрендерить как выпадающий список 

для каждой борды можно выставлять свои настройки 

лучше всего создать отдельный лист std::list<hackrf_device*>, и с каждым из них ассоциировать свою 
структуру HRFParams, которую надо получать из открытого фрейма 
Для каждой борды рендерить свой frame 
frame должен запоминать свои настройки, чтобы не сбивать пользователя с толку 
лучше добавить сбоку диалоговое окно, которое будет "озвучивать" действия пользователя и выводить 
прочую полезную инфу
